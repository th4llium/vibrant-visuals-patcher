# Security Policy

## Supported Versions

Currently, the following versions of `vibrant-visuals-patcher` are actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

If you believe you have found a security vulnerability in `vibrant-visuals-patcher`, please report it by sending an email to:
**[69chubbs@gmail.com](mailto:69chubbs@gmail.com)**

Please include the following details in your report:
- Description of the location and potential impact of the vulnerability.
- A detailed description of the steps required to reproduce the vulnerability (POC scripts/code are highly appreciated).
- Any Minecraft versions, mod versions, or OS environment details needed to reproduce it.

You should receive a response within 48 hours acknowledging receipt of your email. We will investigate the issue and work with you to patch the vulnerability and safely coordinate a release.
